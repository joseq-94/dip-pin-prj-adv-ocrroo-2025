# User Stories — OCR Web Application

## 1. Upload and Extract
I am working as an office worker, I want to upload an image so that 
I can extract text quickly and avoid manual typing.

## 2. Preview OCR Result
I am a student, I want to preview the extracted text so that 
I can correct mistakes before downloading it.

## 3. Personalization
As a frequent user, I want the interface to remember my theme preference so that 
I feel comfortable using the app every day.
