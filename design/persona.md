# Persona — OCR Web Application

## Name
Jose Smith

## Age
25

## Occupation
Administrative Assistant at a small logistics company

## Technical Level
Intermediate – comfortable with basic web tools but not technical

## Goals
- Extract text from images quickly without installing software
- Reduce manual typing from scanned documents
- Save time preparing reports and emails

## Frustrations
- OCR tools that are too complex or require many steps
- Low accuracy when processing receipts or photos
- Interfaces that do not work well on laptops with small screens

## Needs
- A simple interface with one main action: upload → extract text
- Clear preview of the OCR result
- Ability to copy or download the extracted text
- A comfortable UI (dark/light mode)

## How our system helps
This OCR web app provides a clean, minimal interface where Sarah can upload an image, preview the extracted text, and download it instantly. The UI adapts to her preferences (dark/light mode) and validates inputs to avoid errors.
